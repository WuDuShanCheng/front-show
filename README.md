## 🚅 安装
#### 前端

``` bash 
 # 下载数据包
 npm install 
 # 运行项目 默认端口8080 
 npm run serve 
 ``` 
 
 #### 后端
``` bash 
 # 建立名为shan_cheng的数据库 数据库文件在shan_cheng_service项目中
 # 修改config.js中的数据库用户名和密码
 # 下载数据包
 npm install 
 # 运行项目 默认端口3000
 npm run start
 ```
